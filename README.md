# 🌍 Global Freelancers Analysis Dashboard

## 📌 Project Overview

This project analyzes a Global Freelancers dataset to understand freelancer demographics, skills, pricing patterns, ratings, and client satisfaction. The goal is to discover valuable insights that can help freelance platforms attract top talent, improve user experience, and support better business decisions.

The project was completed using Python for data cleaning, exploratory data analysis (EDA), and visualization, while Power BI was used to build an interactive dashboard.

---

## 🎯 Business Problem

Freelance platforms need to attract and retain high-quality freelancers while ensuring a positive experience for clients.

This project aims to answer questions such as:

- Which freelancer age groups perform best?
- What skills are most commonly available?
- Which countries have the highest freelancer concentration?
- How do hourly rates relate to freelancer ratings?
- What factors can improve client satisfaction?

The insights can help freelance platforms optimize their talent acquisition strategy and improve platform efficiency.

## 🔍 Key Insights

### 1. Age Distribution

The majority of freelancers belong to the 20–30 age group, indicating a large and active young workforce.

### 2. Rating vs Hourly Rate

Some freelancers with very low or zero ratings were found charging high hourly rates. This could impact client trust and hiring decisions.

### 3. Geographic Concentration

Certain countries have a significantly higher number of freelancers, making them strong talent hubs for remote work.

### 4. Skill Availability

A few core skills dominate the marketplace, showing areas where talent availability is highest.

### 5. Client Satisfaction

Freelancer performance varies across age groups, providing useful information for talent selection and project assignments.
